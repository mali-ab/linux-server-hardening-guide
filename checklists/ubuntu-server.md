# Ubuntu Server Hardening Checklist

- [ ] Create a non-root admin user
- [ ] Configure `sudo` access
- [ ] Install security updates
- [ ] Configure UFW or nftables
- [ ] Harden SSH settings
- [ ] Install and configure Fail2ban or CrowdSec
- [ ] Enable unattended-upgrades
- [ ] Configure centralized logging or auditd
- [ ] Set up backups and test restores
- [ ] Review users and groups
