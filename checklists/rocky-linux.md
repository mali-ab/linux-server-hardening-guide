# Rocky Linux Hardening Checklist

- [ ] Confirm minimal installation
- [ ] Create separate admin user
- [ ] Harden SSH and disable password login
- [ ] Configure firewalld or nftables rules
- [ ] Enable automatic security patches
- [ ] Install auditd and configure logging
- [ ] Restrict sudo access and service accounts
- [ ] Secure container runtime if using Docker
- [ ] Configure backup strategy
- [ ] Review incident response procedures
