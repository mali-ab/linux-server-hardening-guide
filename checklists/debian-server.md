# Debian Server Hardening Checklist

- [ ] Update package lists and upgrade packages
- [ ] Create and validate SSH keys
- [ ] Configure `sudo` for admin users
- [ ] Harden SSH and disable root login
- [ ] Configure firewall rules
- [ ] Install auditd and log monitoring
- [ ] Enable unattended security updates
- [ ] Use strong TLS settings for web services
- [ ] Audit system accounts and services
- [ ] Document recovery procedures
